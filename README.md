# BookVerse

Aplicación hecha con React Native, Expo y NativeWind.

## Funcionalidades
- Registro
- Inicio y cierre de sesión
- Actualización de datos personales
- Listado de productos
- Crear producto
- Editar producto
- Subida de imagen a Cloudinary y guardado solo de la URL

## Variables de entorno
Crea o edita el archivo `.env`:

```env
EXPO_PUBLIC_API_URL=https://ecommerce-api.wittysky-ae597b7e.westus2.azurecontainerapps.io
EXPO_PUBLIC_CLOUDINARY_CLOUD_NAME=TU_CLOUD_NAME
EXPO_PUBLIC_CLOUDINARY_UPLOAD_PRESET=TU_UPLOAD_PRESET
```

## Ejecutar
```bash
npm install
npx expo start
```
